CSS Snippet Cheatsheet

Ever have trouble recalling the exact syntax for your favorite CSS code? Give it a permanent home and add it to this page! Select any snippet from below and it'll automatacly select all of the code for you to copy.

Flexbox Row

Use these three properties to create a Flexbox row layout.

        .row {
            display: flex;
            flex-direction: row;
            flex-wrap: wrap;
        }
Flexbox Column

Use this to create a Flexbox column layout.

            .column {
                display: flex:
                flex-direction: column
            }
